## About

Supabase Starter is a simple web-app boilerplate with Supabase auth, Upstash Redis and Once UI. It includes email authentication and a profile page to manage user data through secure API routes.

## Quick start

Clone the git repository:

<CodeBlock compact marginBottom="16" codes={[
  {
    code: "git clone https://github.com/once-ui-system/supabase-starter.git",
    language: "bash"
  }
]} />

Install the necessary dependencies:

<CodeBlock compact marginBottom="16" codes={[
  {
    code: "npm install",
    language: "bash"
  }
]} />

Start the local development server:

<CodeBlock compact marginBottom="16" codes={[
  {
    code: "npm run dev",
    language: "bash"
  }
]} />

For the complete setup, please follow the steps in the `README.md` file.
