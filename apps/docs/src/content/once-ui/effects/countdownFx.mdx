The `CountdownFx` component wraps `CountFx` to create beautiful countdown timers. It automatically calculates and displays the time remaining until a target date with smooth wheel animations and colon separators.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={<CountdownFxBasic />}
  codes={[
    {
      code:
`<CountdownFx
    targetDate={new Date(Date.now() + 24 * 60 * 60 * 1000)}
    variant="display-strong-l"
    format="HH:MM:SS"
/>`,
      language: "tsx",
      label: "CountdownFx"
    }
  ]}
/>

## On Complete Callback

Execute a function when the countdown reaches zero.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={<CountdownFxOnComplete />}
  codes={[
    {
      code:
`<CountdownFx
    targetDate={new Date(Date.now() + 5000)}
    format="MM:SS"
    onComplete={() => {
      console.log('Countdown complete!');
      // Trigger your action here
    }}
/>`,
      language: "tsx",
      label: "On Complete"
    }
  ]}
/>
