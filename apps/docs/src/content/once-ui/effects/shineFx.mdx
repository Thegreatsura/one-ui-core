ShineFx creates a smooth, animated shine effect that sweeps across text. The effect uses a gradient that moves horizontally, creating a shimmering appearance ideal for drawing attention to important text.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <ShineFx variant="heading-default-xl" speed={0.75}>
      Thinking...
    </ShineFx>
  }
  codes={[
    {
      code:
`<ShineFx variant="heading-default-xl">
    Shine bright like a diamond
</ShineFx>`,
      language: "tsx",
      label: "Basic"
    }
  ]}
/>

## Speed control

Adjust the animation speed with the `speed` prop. Higher values create slower animations, while lower values make the shine effect faster.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={

        Fast shine (2s)
      </ShineFx>
      <ShineFx variant="heading-default-l" speed={5}>
        Default shine (5s)
      </ShineFx>
      <ShineFx variant="heading-default-l" speed={10}>
        Slow shine (10s)
      </ShineFx>
    </Column>
  }

    Fast shine (2s)

</ShineFx>

<ShineFx speed={10}>
    Slow shine (10s)
</ShineFx>`,
      language: "tsx",
      label: "Speed"
    }
  ]}
/>

## Base opacity

Control the base text visibility with the `baseOpacity` prop (0-1). Higher values improve readability and accessibility, while lower values create a more subtle effect.
