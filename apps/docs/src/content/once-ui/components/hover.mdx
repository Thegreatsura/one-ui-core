The `Hover` component wraps any element and displays an overlay on top when the user hovers over it or focuses on it. It's perfect for creating interactive cards with hover effects, image overlays, or any content that needs to reveal additional information on interaction.

**Touch behavior:** By default (`touch="disable"`), the hover effect is disabled on touch devices. You can set `touch="enable"` to allow hover interactions, or `touch="display"` to always show the overlay on touch devices.

## Accessibility

The component supports keyboard navigation by default. When `showOnFocus` is enabled (default), the overlay will appear when the trigger receives focus, making it accessible for keyboard users.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Row fillWidth horizontal="center">
      <Hover
        fillWidth
        style={{ maxWidth: '24rem' }}
        showOnFocus={true}
        trigger={
          <Card padding="l" direction="column" gap="12">
            <Text variant="heading-strong-m">Keyboard accessible</Text>
            <Text onBackground="neutral-weak" variant="body-default-s">
              Tab to this card to see the overlay with keyboard navigation.
            </Text>
          </Card>
        }
        overlay={
          <Flex
            fillWidth
            fillHeight
            background="rgba(100, 50, 200, 0.8)"
            align="center"
            justify="center"
          >
            <Text variant="heading-strong-l" style={{ color: 'white' }}>
              Focused!
            </Text>
          </Flex>
        }
      />
    </Row>
  }
  codes={[
    {
      language: "tsx",

## API reference

<PropsTable
  content={[
    ["trigger", "ReactNode", "The element to wrap (e.g., Card, Image)"],
    ["overlay", "ReactNode", "Content to display on hover/focus"],
    ["delay", "number", "Delay in milliseconds before showing the overlay (default: 0)"],
    ["hideDelay", "number", "Delay in milliseconds before hiding the overlay (default: 0)"],
    ["disabled", "boolean", "Disable hover functionality (default: false)"],
    ["showOnFocus", "boolean", "Show overlay on keyboard focus (default: true)"],
    ["touch", "'disable' | 'enable' | 'display'", "Touch device behavior: 'disable' (default) - no hover, 'enable' - allow hover, 'display' - always show overlay"],
    ["overlayClassName", "string", "Custom className for the overlay"],
    ["overlayStyle", "React.CSSProperties", "Custom inline styles for the overlay"],
    ["...flex"]
  ]}
