An interactive carousel that uses native CSS scroll-snap for smooth, performant scrolling. Features native touch/swipe gestures on mobile and drag-to-scroll on desktop for seamless navigation between slides.

## Interactions

The Swiper uses native CSS scroll-snap, providing smooth touch gestures on mobile, trackpad scrolling, and mouse drag-to-scroll on desktop. Simply click and drag to navigate between slides.

## Controls

The `controls` prop supports three options: `true` (appear on hover), `false` (hidden), or `"contained"` (always visible inside the carousel). Buttons are automatically hidden when at the first or last slide.

<CodeBlock
  marginTop="16"
  marginBottom="24"
  preview={
    <Column gap="32" fillWidth>
      <Swiper
        controls={true}
        items={[
          { slide: "/images/docs/once-ui/vibe-coding-light.jpg", alt: "Vibe coding light theme" },
          { slide: "/images/docs/once-ui/vibe-coding-dark.jpg", alt: "Vibe coding dark theme" },
